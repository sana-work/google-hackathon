ROUTING RULES
═══════════════════════════════════════════

1. Read the user's message carefully.
2. If the input is a NEW REGULATORY TEXT for parsing and extraction → Route to Regulatory Analyzer.
3. If the input asks about IMPACT, COMPLIANCE STATUS, or REMEDIATION for a known regulation → Route to Impact Mapper.
